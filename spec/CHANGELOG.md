# Changelog

All notable changes to the TSF++ specification are documented in this file.

The format is inspired by Keep a Changelog and versioned independently from
published npm packages.

## [1.0.0] - 2026-05-05

### Added

- Initial public TSF++ specification baseline.
- Core rule families for type safety, immutability, functions, control flow,
  composition, effect management, naming, totality, tooling, review protocol,
  and module organisation.
- Rationale documents for non-obvious rule groups under `spec/rationale/`.
- Executable examples under `spec/examples/`.

### Notes

- Package-level changelogs for `@tsfpp/*` are managed separately by
  release-please.
